\section*{FICHE DE VALIDATION DU LOGICIEL MASCARET V7P0}

\subsection*{Validation du noyau de calcul transitoire}

\subsection*{\emph{Onde de détente dans un canal rectangulaire.}}

\subsection*{Numéro du cas test : 12}

\subsection*{Auteur : N.GOUTAL}


\vspace{1cm}

\subsection*{Description}

Ce cas test a pour but de valider le noyau de calcul REZO dans le cas d'une onde de détente dans un canal rectangulaire. Le domaine de calcul $(x,t)$ est tel que la condition limite aval n'a pas d'influence (canal infini).

\subsection*{Données géométriques}

Le calcul est réalisé dans un canal rectangulaire de pente nulle, de longueur $18\mbox{ } km$ et de largeur $200\mbox{ } m$. La géométrie du canal est décrite par 2 profils en travers situés aux abscisses $X = 0\mbox{ }m$ et $X = 18000\mbox{ } m$.

\subsection*{Données physiques}

Le frottement n'est pas pris en compte.

\subsection*{Données numériques}

Le domaine a été divisé en 200 mailles de longueur constante égale à $90\mbox{ } m$.\\ 
Le pas de planimétrage est homogène dans le domaine égal à $0.25\mbox{ } m$ (40 pas de planimétrage). \\
Le pas de temps est de $10\mbox{ } s$ et le calcul est mené pendant $1000\mbox{ } s$ soit 100 pas de temps.
