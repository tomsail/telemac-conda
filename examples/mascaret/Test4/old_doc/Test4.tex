\section*{FICHE DE VALIDATION DU LOGICIEL MASCARET V7P0}

\subsection*{Validation des noyaux de calcul permanent}

\subsection*{\emph{Canal rectangulaire en lit composé (lit mineur et lit majeur) avec une zone de stockage}}

\subsection*{Numéro du cas test : 4}

\subsection*{Auteur : N. Goutal}


\vspace{1cm}

\subsection*{Description}

	Ce cas test a pour but de valider le noyau de calcul en régime permanent, dans le cas d'un canal rectangulaire en lit composé (lit mineur et lit majeur), avec une zone de stockage.

\subsection*{Données géométriques}

	Le calcul est réalisé dans un canal de pente nulle, de longueur $4990\mbox{ } m$, dont chaque section en travers est de forme rectangulaire de $1\mbox{ } m$ de large. La géométrie du canal est décrite par 2 profils en travers situés aux abscisses $X = 0\mbox{ } m$ et $ X =4990\mbox{ } m$.
La répartition entre lit mineur, lit majeur et zone de stockage est la suivante :

 \end{center}

\subsection*{Données physiques}

Prise en compte du frottement : oui
\begin{itemize} 
 \item Coefficient de Strickler:
 \begin{itemize} 
 \item lit mineur : $90\mbox{ } m^{1/3}.s^{-1}$
 \item lit majeur : $20\mbox{ } m^{1/3}.s^{-1}$
 \end{itemize}
 \item Conditions aux limites : 
 \begin{itemize}
  \item imposée à l'aval égale à $1\mbox{ } m$
  \item Débit imposé à l'amont constant égal à $1\mbox{ } m^3.s^{-1}$
  \end{itemize}
 \item Conditions initiales : aucune 
\end{itemize} 
 


\subsection*{Données numériques}

	Le domaine a été divisé en 499 mailles de longueur constante égale à $10 \mbox{ } m$.
Le pas de planimétrage est homogène dans le domaine égal à $10 \mbox{ } m$ (2 pas de planimétrage).


\subsection*{Solution analytique}

	La répartition des débits se calcule sur la base d'un modèle "DEBORD". 
Ainsi,


\begin{equation}
 Q_{2}=\frac{Q}{1+\eta}
\end{equation}
\\

et
\\
\begin{equation}
 Q_{1}=Q-Q_{2}
\end{equation}

\vspace{0.3cm}

avec


\vspace{0.3cm}

\begin{equation}
\eta =\frac{DEB1}{DEB2}
\end{equation}

\vspace{0.3cm}

\begin{equation}
DEB1=\frac{1}{2}K_{1}ALh^{5/3}
\end{equation}

\vspace{0.3cm}

\begin{equation}
DEB2=\frac{1}{2}K_{2}L\sqrt{\left(\frac{3}{10}\right)^2+\frac{3}{20}\left(1-A^{2}\right)}h^{5/3}
\end{equation}
\\

(d'après la répartition à part égale entre lit mineur et lit majeur)

\vspace{0.3cm}

\begin{equation}
A=0.9\left(\frac {K_{2}}{K_{1}}\right)^{1/6}
\end{equation}

\vspace{0.3cm}

Application numérique :
\\
$K_{1} = 90$ 
\\
$K_{2} = 20$
\\
$L =1\mbox{ } m$
\\
$h = 1 \mbox{ } m$

\vspace{0.3cm}

Ces valeurs donnent $Q_{2} = 0.2056 \mbox{ } m^3.s^{-1}$ et $Q_{1} = 0.7944\mbox{ }  m^3.s^{-1}$.


\subsection*{Résultats}

	La figure~\ref{fig1} présente la ligne d'eau obtenue avec la version MASCARET V7P0. D'autre part, les débits calculés correspondent aux valeurs analytiques puisque l'on trouve $0.794\mbox{ }m^3.s^{-1}$ dans le lit mineur et $0.206\mbox{ } m^3.s^{-1}$ dans le lit majeur.

\subsection*{Conclusion}

	Le traitement de la répartition lit mineur / lit majeur avec zone de stockage semble donc correct. D'autre part, les écarts entre les deux versions sont quasiment nuls.
Le noyau de calcul permanent peut donc être validé dans le cas d'un canal rectangulaire, en lit composé (lit mineur et lit majeur), avec une zone de stockage.
