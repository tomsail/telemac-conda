\section*{FICHE DE VALIDATION DU LOGICIEL MASCARET V7P0}

\subsection*{Validation des noyaux de calcul permanent et transitoire en régime transcritique}

\subsection*{\emph{Canal rectangulaire en lit composé (lit mineur et lit majeur)}}

\subsection*{Numéro du cas test : 3}

\subsection*{Auteur : N. Goutal}


\vspace{1cm}

\subsection*{Description}

	Ce cas test a pour but de valider le noyau de calcul en régime permanent et le noyau transitoire en régime transcritique, dans le cas d'un canal rectangulaire en lit composé (lit mineur et lit majeur) et de s'assurer de la non régression avec la version 5.0.

\subsection*{Données géométriques}

	Le calcul est réalisé dans un canal de pente nulle, de longueur $4990\mbox{ } m$, dont chaque section en travers est de forme rectangulaire de $1\mbox{ } m$ de large. La géométrie du canal est décrite par 2 profils en travers situés aux abscisses $X = 0\mbox{ } m$ et$ X =4990\mbox{ } m$. La répartition entre lit mineur et lit majeur est la suivante :

\subsection*{Données physiques}

Prise en compte du frottement : oui
\begin{itemize} 
 \item Coefficient de Strickler:
 \begin{itemize} 
 \item lit mineur : $90\mbox{ } m^{1/3}.s^{-1}$
 \item lit majeur : $20\mbox{ } m^{1/3}.s^{-1}$
 \end{itemize}
 \item Conditions aux limites : 
 \begin{itemize}
  \item imposée à l'aval égale à $1\mbox{ } m$
  \item Débit imposé à l'amont constant égal à $1\mbox{ } m^3.s^{-1}$
  \end{itemize}
 \item Conditions initiales : aucune pour le noyau permanent
 \begin{itemize}
  \item Débit constant égal à $1\mbox{ } m^3.s^{-1}$ et cote variant linèairement de 1 à 2 $m$ pour les noyaux transitoires. 
  \end{itemize}
\end{itemize} 
 


\subsection*{Données numériques}

	Le domaine a été divisé en 499 mailles de longueur constante égale à $10 \mbox{ } m$.
Le pas de planimétrage est homogène dans le domaine égal à $10 \mbox{ } m$ (2 pas de planimétrage).


\subsection*{Solution analytique}

	La répartition des débits se calcule sur la base d'un modèle "DEBORD". 
Ainsi :


\begin{equation}
 Q_{2}=\frac{Q}{1+\eta}
\end{equation}
\\
et\\
\\
\begin{equation}
 Q_{1}=Q-Q_{2}
\end{equation}
\\
avec\\
\\
\begin{equation}
\eta =\frac{DEB1}{DEB2}
\end{equation}

\vspace{0.3cm}

\begin{equation}
DEB1=\frac{1}{2}K_{1}ALh^{5/3}
\end{equation}

\vspace{0.3cm}

\begin{equation}
DEB2=\frac{1}{2}K_{2}L\sqrt{2-A^{2}}h^{5/3}
\end{equation}
\\

(d'après la répartition à part égale entre lit mineur et lit majeur)

\vspace{0.3cm}

\begin{equation}
A=0.9\left(\frac {K_{2}}{K_{1}}\right)^{1/6}
\end{equation}

\vspace{0.3cm}

Application numérique :
\\
$K_{1} = 90$ 
\\
$K_{2} = 20$
\\
$L =1\mbox{ } m$
\\
$h = 1 \mbox{ } m$

\vspace{0.3cm}

Ces valeurs donnent $Q_{2} = 0.2805\mbox{ } m^3.s^{-1}$ et $Q_{1} = 0.7195\mbox{ }  m^3.s^{-1}$.


\subsection*{Résultats}

	La figure~\ref{fig1} compare les lignes d'eau obtenues avec le noyau permanent et le noyau transcritique (schéma implicite et explicite). Les résultats sont cohérents. D'autre part, les débits calculés avec le noyau permanent correspondent aux valeurs analytiques puisque l'on trouve $0.72 \mbox{ } m^3.s^{-1}$ dans le lit mineur et $0.28\mbox{ } m^3.s^{-1}$ dans le lit majeur.
Le calcul convergé  du noyau transitoire implicite donne une erreur de l'ordre de $5 \% $ à l'aval du modèle. Pour l'ensemble des calculs, les maillages sont identiques.
