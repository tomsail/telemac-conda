med_int MEDsupportMeshnAxis ( const med_idt  fid,
const int  meshit 
)

Cette routine permet de lire dans un maillage support le nombre d'axes du repère des coordonnées des noeuds.

Paramètres
fidIdentificateur du fichier.
meshitItérateur sur les maillages. Cet itérateur commence à 1.
Valeurs retournées
med_intNombre d'axe dans le repère de coordonnées.

Cette routine permet de lire dans un maillage support le nombre d'axes du repère des coordonnées des noeuds. le nombre d'axe correspond à la dimension de l'espace de calcul. L'accès au maillage support se fait via un itérateur.
