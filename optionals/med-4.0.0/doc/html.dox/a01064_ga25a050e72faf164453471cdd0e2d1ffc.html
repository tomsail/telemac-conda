subroutine mfiexi ( character *(*)  fname,
integer  access,
integer  fexist,
integer  accok,
integer  cret 
)

Interroge l'existence d'un fichier de nom filename et la possibilité de l'ouvrir selon le mode d'accès accessmode.

Paramètres
fnameNom du fichier.
accessMode d'acces au fichier.
fexistindique l'existence d'un fichier.
accokindique l'adéquation des droits effectifs avec le mode d'accès demandé .
cretretour négatif en cas d'erreur, Zéro sinon.
Voir également
MEDfileExist

Interroge l'existence d'un fichier de nom filename et la possibilité de l'ouvrir selon le mode d'accès accessmode. Si le fichier n'existe pas, cette fonction prend en compte les droits du répertoire contenu dans le chemin d'accès filename pour savoir s'il est possible de créer le fichier.
