Découvrir les concepts et fonctionnalités avancés



Tous les types et paramètres utiles à l'utilisation de l'API de la bibliothèque MED sont définis dans les fichiers med.h (API C) et med.hf (API FORTRAN). Les deux API donnent accès aux mêmes fonctionnalités.

Pour aborder ce chapitre, tous les concepts et fonctionnalités présentés dans le chapitre 2 (Concepts et fonctionnalités de base) sont supposés connus.
