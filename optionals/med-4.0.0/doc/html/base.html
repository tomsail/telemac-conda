MED 3.0 et MED 2.x

Les routines de l'API MED 3.0 présentées dans ce guide d'utilisation, permettent de lire un fichier MED 2.x, mais elles ne permettent pas de le modifier (écriture impossible).

La bibliothèque MED est livrée avec un utilitaire medimport qui permet de convertir un fichier 2.x au format 3.0.

Pour modifier ou ajouter de nouvelles données dans un fichier MED 2.x sans le convertir, il est nécessaire d'employer les routines de l'API 2.3.
