Ce guide d'utilisation permet à l'utilisateur de la bibliothèque MED (Modélisation et Echange de Données) de réaliser une interface d'écriture et de lecture de fichier au format MED dans un code de calcul. Ce guide peut être vu comme un tutoriel avec de nombreux cas et conseils d'utilisation.

La découverte de l'utilisation de la bibliothèque MED se veut progressive. Après une rapide introduction présentant le modèle d'échanges de données et la bibliothèque, le chapitre 2 permet de découvrir les fonctionnalités de base, le chapitre 3 présente les concepts et les fonctionnalités avancés. L'usage de ces fonctionnalités avancées dépend des capacités de chaque code de calcul à pouvoir les exploiter.
